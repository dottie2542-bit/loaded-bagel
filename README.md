## My Task Tracker!

### Very simple code I created in order to practice. You can use it in VScode!

> MAKE SURE YOU have PYTHON INSTALLED AS WELL
##
<p>To use it, you have to create a folder in your File Explorer. Call it "my_task_tracker".</p>
<p>You should then open up VScode and press "Open Folder".</p>
<p>Select the folder "my_task_tracker" and then create a new file called "task_cli.py"</p>
<p>Paste the code in the box!</p>
<p>Once it's in, open a new terminal on the top of VScode and run this command to add your first task!</p>

`python task_cli.py add "Your Task Here"`

A `tasks.json` file should appear, showing what you have to do!
